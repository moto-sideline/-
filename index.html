📖 覚醒の儀式（APIキーの取り方）

※ API取得ページは別タブで開きます。このアプリのタブは閉じずに戻ってきてください。

  1. 下のボタンから Google AI Studio を開きます。
  2. Googleアカウントでログインします。
  3. 「Create API Key」でキーを発行し、コピーします。
  4. 【⚙️設定】に貼り付けて「保存して覚醒」を押します。
