# Kindle執筆チェックリスト集

---

## 🎯 Kindle執筆での活用法

このナレッジは、Kindle執筆の**全工程で品質管理**に活用できます：

| 活用場面 | 具体的な使い方 |
|---|---|
| **企画段階** | 第1部「ビジネスカルテ」でKindle本の方向性を整理 |
| **キャラクター設定** | 第2部「キャラクター設定」で著者としての一貫性を確保 |
| **タイトル作成** | 第3部「パワーワード選定」で売れるタイトルを作成 |
| **執筆前** | 第4部「執筆前チェック」で準備漏れを防止 |
| **執筆中** | 第5部「文章品質チェック」で読みやすさを確保 |
| **執筆後** | 第6部「全体構成チェック」で完成度を確認 |
| **出版前** | 第7部「Amazon商品ページ準備」で販促準備 |
| **出版後** | 第8部「成果測定」で改善点を把握 |

**ポイント**: このチェックリストは**Kindle出版の全工程**をカバーしています。特に**第4部「執筆前チェック」**と**第6部「執筆後チェック」**は、品質を担保するために必ず使いましょう。

## 第4部：執筆前チェックリスト

### 4.1 企画段階チェックリスト

```
【Kindle本企画チェック】

□ テーマは明確か
□ ターゲット読者は具体的か
□ 解決する問題は明確か
□ 競合書籍を3冊以上調査したか
□ 差別化ポイントは明確か
□ 独自の価値提供があるか

【市場性チェック】
□ 需要は確認できているか
□ 市場規模は十分か
□ トレンドに合っているか
□ 長期的に価値があるか

【実現可能性チェック】
□ 必要な知識・経験はあるか
□ エビデンスは集められるか
□ 執筆時間は確保できるか
□ 予算は問題ないか
```

### 4.2 構成チェックリスト

```
【目次構成チェック】

□ 章立ては論理的か
□ 各章のバランスは適切か
□ 4つの学習タイプに対応しているか
　□ なぜ型の要素
　□ なに型の要素
　□ どうやって型の要素
　□ いますぐ型の要素

【各章のチェック】
□ 導入は興味を引くか
□ 本論は分かりやすいか
□ 結論は明確か
□ 次章への橋渡しはあるか
□ 実践課題は設定されているか
```


## まとめ

本ワークシート集は、Kindle出版を成功に導くための実践的なツールです。各シートを順番に埋めていくことで、企画から執筆、出版、改善まで、体系的かつ効率的に進めることができます。

単にチェックリストとして使うのではなく、思考を整理し、アイデアを具体化するツールとして活用してください。定期的に見返し、更新することで、あなたのKindle出版スキルは確実に向上していくでしょう。
