# 『続・騙されない為の最低限の事柄』表紙デザイン指示書

デザイナー（またはご自身でCanva等で作成する際）に渡すためのコンセプト・プロンプトシートです。

## 1. 全体コンセプトとトーン＆マナー
*   **前作との統一感**：ベースはミントグリーン（薄緑）、下部に太いブラウンの帯。前作の続編であることが一目でわかるようにする。
*   **メインテーマ**：「怪しいパーカーの人物の正体（からくり・自動化の罠）」を暴く。
*   **雰囲気**：フラットデザイン・シルエット中心で、少し不気味だがキャッチーでわかりやすいトーン。

## 2. 構図とイラスト要素
*   **背景**：上部3/4がミントグリーン、下部1/4がブラウン。
*   **人物の配置**：前作のように、パソコンを触る「パーカー・フードを被った匿名人物」が複数並んでいる。
*   **中心の人物（主役）**：真ん中の一番目立つ人物は、**仮面（フードの顔部分）がバカッと外（または割れ）れており、頭の中が「機械の歯車」になっている**デザイン。（からくりや全自動システムの暗喩）
*   **遊び心（オプション）**：「設計図」や「職人のコテ」などを、さりげなくアクセントとして配置しても面白い。

## 3. テキスト配置（タイポグラフィ）
*   **メインタイトル**：『続・騙されない為の最低限の事柄』
    *   **「続」**の文字を赤色で一番大きく、アイキャッチとして配置。
    *   文字のフォントは、前作と同じく「少し手書き風で親しみやすいゴシック体」。
*   **サブタイトル**：『情報商材の裏側にある「設計図」をすべて見せます』
    *   ブラウンの帯の中、もしくはタイトルの周辺に配置し、引き締める。
