# 付録1：セールスレター・スキャニング・シート

レターを読みながら、以下の項目に当てはまるものがないかチェックしてみてください。
3つ以上当てはまる場合は、その場で購入ボタンを押さず、一度ブラウザを閉じて工房の空気を吸いに行くことをお勧めします。

## 【視覚的な仕掛け】
- [ ] **残り時間のカウントダウンが動いている**
    （ページを再度読み込んでもリセットされるなら、それは単なる画像やプログラムです）
- [ ] **「あと〇名」という残席表示が頻繁に更新される**
    （焦りを生むための演出部品である可能性が高いです）
- [ ] **派手な実績画像（銀行通帳や札束）が並んでいる**
    （職人の世界で「成果物」を見せずに「お金」だけを見せるのは異常です）

## 【言葉の罠】
- [ ] **「初心者でも」「スマホ一台で」「コピペで」という言葉がある**
    （物作りにおいて「簡単に」という言葉は、大抵「手抜き」か「嘘」です）
- [ ] **「今だけ限定」「今すぐ買わないと後悔する」と煽っている**
    （本物の価値は、明日になっても逃げません）
- [ ] **具体的な作業内容（HTML、サーバー、学習量）がボカされている**
    （「型」という設計図を動かすための重装備が隠されています）

## 【構造の罠】
- [ ] **「よくある質問」で、あなたの不安が都合よく解消されている**
    （それは対話ではなく、袋小路への誘導です）
- [ ] **特定商取引法に基づく表記の住所が、バーチャルオフィスや空き家である**
    （逃げ道を用意している販売者の典型的な特徴です）
- [ ] **サポート期間が3か月〜6か月と短く設定されている**
    （あなたが挫折した頃に、連絡を断つための「逃げの期限」です）

> [!TIP]
> **職人からのアドバイス**
> このシートは、あなたを「否定」するためのものではありません。
> 感情が大きく揺さぶられたときに、一ミリでも「冷静な自分」を取り戻すための、いわば**水平器（レベル）**のようなものです。
