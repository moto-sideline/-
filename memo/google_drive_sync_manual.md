# 魔法のランプ - Google Drive 自動同期設定マニュアル（ハードユーザー向け）

このマニュアルは、PCとスマートフォン間で「魔法のランプ」のデータを自動的に同期させたいハードユーザー向けの手順書です。

「魔法のランプ」には専用のサーバー（バックエンド）が存在しません。
そのため、あなた自身の「Google Drive」をデータの保存先として利用し、そこにバックアップデータを自動で読み書きする仕組みを使用します。

この機能を有効にするには、Google Cloudで「プロジェクト」を作成し、専用の「クライアントID」をご自身で取得する必要があります。少し専門的な手順になりますが、以下のステップに沿って進めてください。

---

## ステップ1：Google Cloud プロジェクトの作成

1. ブラウザで [Google Cloud Console](https://console.cloud.google.com/) にアクセスし、Googleアカウントでログインします。
2. 画面上部のプロジェクト選択ドロップダウン（「プロジェクトの選択」または現在のプロジェクト名）をクリックします。
3. 右上の「新しいプロジェクト」をクリックします。
4. プロジェクト名に「`Magic Lamp Sync`」などと入力し、「作成」をクリックします。（作成まで数秒〜数十秒かかります）
5. プロジェクトが作成されたら、通知アイコンから作成したプロジェクトを選択するか、再度画面上部のドロップダウンから選択して切り替えます。

---

## ステップ2：Google Drive API の有効化

1. 左側のナビゲーションメニュー（三本線アイコン）から **「API とサービス」** ＞ **「ライブラリ」** を選択します。
2. 検索バーに「`Google Drive API`」と入力して検索します。
3. 検索結果から「**Google Drive API**」をクリックし、「**有効にする**」ボタンをクリックします。

---

## ステップ3：OAuth 同意画面の設定

1. 左側のナビゲーションメニューから **「API とサービス」** ＞ **「OAuth 同意画面」** を選択します。
2. User Type（ユーザーの種類）で「**外部（External）**」を選択し、「作成」をクリックします。（※Google Workspaceを使用している場合は「内部」でも構いません）
3. **アプリ情報**を入力します：
   - アプリ名：「`魔法のランプ`」など
   - ユーザーサポートメール：ご自身のメールアドレス
   - デベロッパーの連絡先情報：ご自身のメールアドレス
4. 「保存して次へ」をクリックします。
5. **スコープ**の画面では何も追加せず、そのまま「保存して次へ」をクリックします。
6. **テストユーザー**の画面で「ADD USERS（ユーザーを追加）」をクリックし、**同期に使用する自分自身のGoogleアカウントのメールアドレス**を入力して追加します。
   - ※これを行わないと、アプリからログインする際にブロックされます。
7. 「保存して次へ」をクリックし、最後に「ダッシュボードに戻る」をクリックします。

---

## ステップ4：OAuth クライアント ID の作成

1. 左側のナビゲーションメニューから **「API とサービス」** ＞ **「認証情報」** を選択します。
2. 画面上部の「**+ 認証情報を作成**」をクリックし、「**OAuth クライアント ID**」を選択します。
3. **アプリケーションの種類** で「**ウェブ アプリケーション**」を選択します。
4. 名前を「`Magic Lamp Web`」などと入力します。
5. **「承認済みの JavaScript 生成元」** の「URI を追加」をクリックし、魔法のランプを公開しているURLを入力します。（末尾のスラッシュは不要です）
   - 例: `https://moto-sideline.github.io`
   - ローカル（PCのフォルダ）から直接開いている場合は `http://localhost` などを入れるか、GitHub Pagesでの運用を推奨します。
6. 「作成」ボタンをクリックします。
7. 画面に **「クライアント ID」**（`xxxxxxxxxxxx-xxxxxxxxxxx.apps.googleusercontent.com` のような文字列）が表示されます。**これをコピーして保存してください。**

---

## ステップ5：魔法のランプへの設定

1. PCまたはスマートフォンの「魔法のランプ」を開きます。
2. 左下の「⚙️（設定アイコン）」をクリックして設定画面を開きます。
3. 「Google Drive 自動同期（ハードユーザー向け）」の項目に、ステップ4で取得した **クライアントID** を貼り付けます。
4. 「保存」ボタンを押します。
5. 設定画面に「Google Drive でログイン」ボタンが表示されますので、クリックしてご自身のGoogleアカウントでログインを許可してください。

これで設定は完了です！

以降は、アプリ起動時にGoogle Driveから最新のデータが自動で読み込まれ、新しく会話をするたびにDriveへ自動保存されるようになります。スマートフォン側でも同じクライアントIDを設定してログインすれば、データの同期が可能です。

> ※この同期データは、あなたのGoogle Driveの隠し領域（Application Data Folder）に保存されるため、通常のドライブ画面からは見えません。アプリからのみアクセスできる安全な領域です。
