# AI出版エンジン

**日々のジャーナルからKindle本を生み出すAIシステム**

---

## このシステムでできること

1. **日刊ジャーナル** → 日々の気づき・学びを蓄積
2. **週次振り返り** → エピソード・信念を自動抽出
3. **Kindle執筆** → 蓄積した素材でKindle本を執筆

```
【あなたがやること】
日々のジャーナルを書くだけ（音声入力OK）

【AIがやること】
・エピソードを抽出
・あなたの文体を学習
・Kindle本の執筆をサポート
```

---

## セットアップ（5分）

### Step 1: Cursorをインストール

1. [Cursor公式サイト](https://cursor.sh/) からダウンロード
2. インストール後、起動

### Step 2: このフォルダをCursorで開く

1. Cursorを起動
2. 「File」→「Open Folder」
3. この「AI出版エンジン」フォルダを選択

### Step 3: プロフィールを設定

`002-Profile/` フォルダ内のファイルを編集：

| ファイル | 内容 |
|---------|------|
| `01_私のコアな価値観.md` | あなたの価値観・信念 |
| `02_実績データベース.md` | あなたの実績・経験 |
| `03_ビジネス設計.md` | あなたのビジネス情報 |

**💡 最初は空欄でもOK。日々のジャーナルから自動的に蓄積されていきます。**

### Step 4: 完了！

これでセットアップ完了です。

---

## 使い方

### 🗓️ 毎日やること（5-10分）

**ジャーナルを書く**

1. `001-Daily-Journal/` フォルダに新しいファイルを作成
2. ファイル名：`YYYY-MM-DD.md`（例：`2024-12-25.md`）
3. その日の気づき・学び・出来事を書く

**💡 音声入力でもOK！思いついたことを話すだけ。**

---

### 📅 週1回やること（10-15分）

**週次振り返りコマンドを実行**

1. Cursorのチャットで以下を入力：

```
/weekly-journal
```

2. AIが自動的に：
   - 1週間分のジャーナルを読み込み
   - エピソード・信念を抽出
   - 週次レポートを作成
   - 使用可能エピソード集に追加

---

### 📚 Kindle本を書くとき

**Kindle作成コマンドを実行**

1. Cursorのチャットで以下を入力：

```
/create-kindle
```

2. AIが自動的に：
   - あなたのエピソード集を参照
   - あなたの文体を学習
   - Kindle本の執筆をサポート

## フォルダ構成

```
AI出版エンジン/
├── .cursor/               ← システム（アップデート対象）
├── Knowledge/             ← システム（アップデート対象）
├── 001-Daily-Journal/     ← あなたのデータ（消さないで！）
├── 002-Profile/           ← あなたのデータ（消さないで！）
├── 003-Kindle-Project/    ← あなたのデータ（消さないで！）
└── Output/                ← あなたのデータ（消さないで！）
```

**⚠️ 重要：** `001-`, `002-`, `003-`, `Output/` フォルダは**あなたのデータ**です。
アップデート時も**絶対に消さないでください**。

## よくある質問

### Q: ジャーナルは何を書けばいいですか？

**A:** 何でもOKです。例えば：

- 今日学んだこと
- 気づいたこと
- 失敗したこと
- 成功したこと
- 読んだ本の感想
- お客様との会話

**ポイント：** 完璧を目指さない。思ったことをそのまま書く。

---

### Q: 音声入力でもいいですか？

**A:** もちろんOKです。むしろ音声の方が自然な言葉で書けます。

iPhoneなら音声入力ボタンを押して話すだけ。
後から整形する必要もありません。

---

### Q: どれくらい続ければKindle本が書けますか？

**A:** 目安として：
- **1ヶ月** → エピソードが10-20個蓄積
- **3ヶ月** → Kindle本1冊分の素材が蓄積

早く書きたい場合は、過去の経験を一気に書き出すのもアリです。

---

### Q: 文体DNAって何ですか？

**A:** あなた独自の文章の特徴です。

- 語尾の癖
- よく使う言い回し
- 文章のリズム

ジャーナルを書き続けると、AIが自動的に学習します。
