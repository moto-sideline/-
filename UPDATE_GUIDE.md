# アップデートガイド

AI出版エンジンを最新版にアップデートする方法です。

---

## ⚠️ 重要：アップデート前の注意

**絶対に消してはいけないフォルダ（あなたのデータ）：**

| フォルダ | 内容 |
|---------|------|
| `001-Daily-Journal/` | あなたが書いたジャーナル |
| `002-Profile/` | あなたの価値観・エピソード |
| `003-Kindle-Project/` | あなたの執筆プロジェクト |
| `Output/` | 出力したファイル |

**これらのフォルダは、どんなことがあっても削除しないでください。**

---

## アップデート手順

### Step 1: バックアップを取る

万が一に備えて、フォルダ全体をコピーしておきます。

```
AI出版エンジン → AI出版エンジン_backup_YYYYMMDD
```

### Step 2: 新しいバージョンをダウンロード

配布された新しいバージョンをダウンロードします。

### Step 3: システムファイルを上書き

**以下のフォルダだけ**を、新しいバージョンで上書きします：

| 上書きするフォルダ | 内容 |
|-------------------|------|
| `.cursor/` | コマンド・エージェント |
| `Knowledge/` | ナレッジファイル |

**上書き方法：**
1. 古いフォルダを削除
2. 新しいフォルダをコピー

### Step 4: 完了確認

Cursorを再起動して、正常に動作するか確認します。

---

## フォルダ構成の説明

```
AI出版エンジン/
├── .cursor/          ← システム（上書きOK）
├── Knowledge/        ← システム（上書きOK）
├── 001-Daily-Journal/  ← あなたのデータ（触らない）
├── 002-Profile/        ← あなたのデータ（触らない）
├── 003-Kindle-Project/ ← あなたのデータ（触らない）
├── Output/             ← あなたのデータ（触らない）
├── README.md         ← 上書きOK
└── UPDATE_GUIDE.md   ← 上書きOK（このファイル）
```

## よくある質問

### Q: 間違えてジャーナルを消してしまった

**A:** バックアップから復元してください。バックアップがない場合は、残念ながら復元できません。**アップデート前のバックアップは必須です。**

### Q: 新しい機能が追加されたけど動かない

**A:** `.cursor/` フォルダが正しく上書きされているか確認してください。Cursorを再起動すると解決することがあります。

### Q: 古いナレッジと新しいナレッジが混在している

**A:** `Knowledge/` フォルダを一度削除してから、新しいバージョンをコピーしてください。
